hce-sticky

A stickily positioned element is an element whose computed position value is sticky. It's treated as relatively positioned until its containing block crosses a specified threshold (such as setting top to value other than auto) within its flow root (or the container it scrolls within), at which point it is treated as "stuck" until meeting the opposite edge of its containing block. (from https://developer.mozilla.org/en-US/docs/Web/CSS/position).

If `position:sticky` satisfy your requirements, it is recommended to use it instead.
