hce-dialog

hce-dialog provides users to show page-blocking modal dialog with title, contents, and buttons. To show call `open` function of `window.hce.dialog` with data; title, contents, actions. Here is the example. 

  window.hce.Dialog.open({
    title: 'Custim Title', 
    contents:'This is custom contents',
    actions: [{
      text: 'Custom Button',
      handler: event => {
        alert('My custom button is clicked');
        window.hce.dialog.close();
      }
    }]
  })

To show the previous dialog, run `window.hce.dialog.open()` without any data.
