hce-tooltip

hce-tooltip is shown on the top of the parent element in default when mouse is over. It is also shown when the parent is focused for accessibility.

Set attribute 'visible' to show the tooltip without user actions; mouseover or focus. You can also set any html inside tooltip.

Attributes

  • visible: Default false. Indicates to show the tooltip when initialized.
  • position: Default top.Where to show the tooltip; left, right, bottom, or top.
