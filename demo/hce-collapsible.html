hce-collapsible

hce-collapsible is a accordion element that expand when clicked on. It allows you to hide content that is not immediately relevant to the user.

Child Elements

  • any element. If "expanded" attribute is given, its body will be shown.
    • hce-header class: Header element of collapsible. It is always visible.
    • hce-body class: Collapsible element when header element clicks.
