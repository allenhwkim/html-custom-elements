hce-calendar

hce-calendar shows month calendar

Attributes

  • selected: Date to show when start. Default is today.
  • min-date: Minimum date to be selectable.
  • max-date: Maximum date to be selectable.
  • weekday-format: "2-letter", "3-letter", or "full".
  • date-format: When selected, formatted date will be returned with `date-selected` event.
  • visible-by: Css selector. When given, calendar works as a dropdown of css selector element.
