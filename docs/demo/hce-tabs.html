hce-tabs

When you click on tab, only contents to matching is visible:w.

Attributes

  • selected: Default ''. What tab contents to be display.

Child Elements

  • "hce-tabs" class in any element: Tab indicator section.
    • "tab" attribute: Tab identifier. String value. e.g., "tab1"
    • "disabled" attribute: If defined, this won't be selected
  • "hce-contents" class in any element: Contents section
    • "tab" attribute: Contents identifier. String value. e.g., "tab1"

To use hce-tabs;

  • Provide two sections with class 'hce-tabs' and 'hce-contents'
  • within 'hce-tabs' section, declare tab with attribute 'tab'
  • within 'hce-contents' section, declare mathing content with attribute 'tab'
