hce-loading

hce-loading shows loading sign for its parent element by blocking parent's contents. You can show and hide loading by calling its method `show()` or `hide()`.

You do not like the default loading sign? No problem, you can override the default by simply proving your own html.

To show loading sign initially, simply set attribute 'loading'.

Attributes

  • loading: If defined loading sign will be displayed covering the parent element.
  • timeout: Timeout seconds. If given, after this passed, `timedout` event will fire.

Events

  • timeout: Fired after 10 seconds from loading sign is shown.
