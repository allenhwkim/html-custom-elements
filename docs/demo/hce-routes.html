hce-routes

hce-routes replace its contents by the current url match.

This is very useful when you need to build a very simple single page application without a framework router.

Child Elements

  • route-match attribute: When the current url matches to this string, ...
  • import attribute: Import the given html file here.
  • default attribute: When the current url does not match any, use this block to import html file.
