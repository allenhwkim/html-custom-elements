hce-routes

hce-routes replace its contents by the current url match.

This is very useful when you need to build a very simple single page application without a framework router.

Attributes

  • src Default url to import. e.g., 'home.html'

Child Element Attributes

  • route-match Regular expression matching to the current url. e.g. 'home', '/#([a-z]+)$'
  • import Import the given html file here. e.g. home.html, {{1}}.html
